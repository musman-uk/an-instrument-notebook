# Roadmap

## v0.0.0 Draft
- Repository structure defined and initial folders created  
- Core layouts established (`index.njk`, `prelude.njk`)  
- Instrument group folders added with the first handwritten style notes  
- Navigation refined with the homepage About link and the return link on the About page  
- Early design notes drafted to guide the handwritten, pinned note aesthetic

- Populate each instrument group with an initial curated set of notes

- Ensure the Eleventy build process reliably generates the static site  
- Prepare the site for gradual expansion with additional notes and instrument groups
